# Android Network Monitoring Tool

## Introduction
This repository contains an Android network monitoring tool designed to help users analyze network traffic and monitor various network activities on their Android devices.

## Setup Instructions
1. **Clone the Repository**  
   Use the following command to clone the repository:  
   ```bash
   git clone https://github.com/COPTAO/COPTAO.git
   ```  

2. **Open the Project**  
   Open the cloned project in Android Studio.

3. **Build the Project**  
   Build the project using the Build menu in Android Studio. You may need to sync Gradle files.

4. **Run the App**  
   Connect your Android device or use an emulator and run the app.

## Permissions Required
- **INTERNET**: Required for accessing network information.
- **ACCESS_WIFI_STATE**: Allows the app to access information about Wi-Fi networks.
- **ACCESS_NETWORK_STATE**: Allows the app to access information about networks.
- **READ_EXTERNAL_STORAGE**: Required for saving logs to external storage.
- **WRITE_EXTERNAL_STORAGE**: Required for writing logs to external storage.

## Features
- Monitor real-time network traffic.
- View detailed logs of network requests.
- Analyze data usage per application.
- Display Wi-Fi network information.
- Export logs to external storage.

## Conclusion
This tool provides valuable insights into network activity and can be beneficial for developers and users who want to monitor network performance.
